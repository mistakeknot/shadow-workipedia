# AC As Class Privilege Thermal Apartheid Emerging

AC ownership 90% but operation unaffordable for 30% → "AC deserts" in low-income neighborhoods (broken window units, no central air) → workplace cooling mandates weak/unenforced → heat exposure = poverty → thermal comfort becomes explicit class marker → surviving summer requires wealth.
